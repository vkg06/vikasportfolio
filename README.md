# Personal Portfolio Website

This is my personal portfolio website created using HTML, CSS, and JavaScript.  
It highlights my skills, projects, and basic information in a simple and responsive design.

## Features

- Responsive design for all screen sizes
- Smooth scrolling and basic animations
- Projects showcase
- Skills and About Me sections
- Contact section

## Technologies Used

- HTML5
- CSS3
- JavaScript
